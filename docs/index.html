Installation

composer require sirv/sirv-image
use Sirv\SirvImage;

Quick Start

$url = (new SirvImage('https://demo.sirv.com/image.jpg'))
    ->resize(300, 200)
    ->quality(80)
    ->format('webp')
    ->toUrl();

Constructor

new SirvImage($urlOrBase, $path = null)

ParamTypeDescription
$urlOrBasestringFull image URL or base URL
$pathstring|nullImage path when using base URL
 
$img = new SirvImage('https://demo.sirv.com/image.jpg');
$img = new SirvImage('https://demo.sirv.com', '/image.jpg');

Resize

MethodDescription
resize(?int $width, ?int $height, ?string $option): selfSet dimensions and scale option (fit, fill, ignore, noup)
width(int $w): selfSet width
height(int $h): selfSet height
scaleByLongest(int $s): selfResize by longest dimension
thumbnail(int $size = 256): selfCreate a square thumbnail
 
$img->resize(400, 300, 'fill')
$img->width(500)
$img->thumbnail(128)

Crop

MethodDescription
crop(?int $width, ?int $height, ?int $x, ?int $y, ?string $type, ?int $padWidth, ?int $padHeight): selfCrop the image. Types: trim, poi, face
clipPath(string $name): selfApply a Photoshop clipping path
 
$img->crop(200, 150, 10, 20)
$img->crop(type: 'face', padWidth: 50)

Rotation

MethodDescription
rotate(float $degrees): selfRotate (-180 to 180)
flip(): selfFlip vertically
flop(): selfFlip horizontally
 
$img->rotate(90)

Format & Quality

MethodDescription
format(string $fmt): selfSet output format: jpg, png, webp, optimal, original
quality(int $q): selfSet JPEG quality (0-100)
webpFallback(string $fmt): selfSet WebP fallback
subsampling(string $value): selfSet chroma subsampling
pngOptimize(bool $enabled = true): selfEnable PNG optimization
gifLossy(int $level): selfSet GIF lossy compression
 
$img->format('webp')->quality(85)

MethodDescription
brightness(int $v): selfAdjust brightness (-100 to 100)
contrast(int $v): selfAdjust contrast (-100 to 100)
exposure(int $v): selfAdjust exposure (-100 to 100)
hue(int $v): selfAdjust hue
saturation(int $v): selfAdjust saturation
lightness(int $v): selfAdjust lightness
shadows(int $v): selfAdjust shadows
highlights(int $v): selfAdjust highlights
grayscale(): selfConvert to grayscale
colorLevel(?int $black, ?int $white): selfSet color levels
histogram(string $channel): selfDisplay histogram: r, g, b

Color Effects

MethodDescription
colorize(?string $color, ?int $opacity): selfApply color overlay
colortone(?string $preset, array $options = []): selfApply color tone preset or custom settings
 
$img->colortone('sepia')
$img->colortone(null, ['color' => 'ff5500', 'level' => 80, 'mode' => 'highlights'])

Effects

MethodDescription
blur(int $v): selfApply blur (0-100)
sharpen(int $v): selfApply sharpening (0-100)
vignette(?int $value, ?string $color): selfApply vignette
opacity(int $v): selfSet opacity (0-100)
 
$img->blur(10)->sharpen(50)

Text Overlay

text(string $content, array $options = []): self

Add a text overlay. Call multiple times for multiple layers.

Option KeyDescription
fontSizeFont size in px
fontFamilyGoogle Fonts family
fontStylenormal, italic
fontWeight300-800
colorText color
opacity0-100
outlineWidthOutline width
outlineColorOutline color
backgroundColorBackground color
alignleft, center, right
positioncenter, north, south, etc.
positionXX offset
positionYY offset
 
$img->text('Hello', ['color' => 'white', 'fontSize' => 24, 'position' => 'center'])
    ->text('World', ['position' => 'south'])

Watermark

watermark(string $path, array $options = []): self

Add a watermark. Call multiple times for multiple watermarks.

Option KeyDescription
positioncenter, north, tile, etc.
scaleWidthWidth (px or %)
scaleOptionnoup, fit, fill, ignore
rotate-180 to 180
opacity0-100
layerfront, back
 
$img->watermark('/wm.png', ['position' => 'center', 'opacity' => 50])

Canvas

canvas(array $options = []): self

Configure canvas. Keys: width, height, color, position, opacity, aspectRatio, borderWidth, borderHeight, borderColor, borderOpacity.

$img->canvas(['width' => 800, 'height' => 600, 'color' => 'ffffff'])
$img->canvas(['aspectRatio' => '16:9'])

Frame

frame(array $options = []): self

Add a frame. Keys: style (solid, mirror, edge, dither), color, width, rimColor, rimWidth.

$img->frame(['style' => 'solid', 'color' => '333333', 'width' => 5])

Other

MethodDescription
page(int $num): selfSelect a PDF page
profile(string $name): selfApply a saved profile

URL Generation

MethodDescription
toUrl(): stringBuild and return the URL
__toString(): stringEnables direct string casting
 
$url = $img->toUrl();
echo $img; // same as toUrl()

Full Examples

E-commerce Product Image

$url = (new SirvImage('https://demo.sirv.com', '/product.jpg'))
    ->resize(800, null, 'fill')
    ->crop(type: 'face')
    ->format('webp')
    ->quality(85)
    ->sharpen(20)
    ->watermark('/logo.png', ['position' => 'southeast', 'opacity' => 40])
    ->text('Sale!', ['color' => 'red', 'fontSize' => 32, 'position' => 'north'])
    ->toUrl();
